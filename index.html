FEATURES

Sharp tools. Zero overhead.

Port Monitor

Scans all TCP LISTEN sockets via lsof. PID, port, protocol, bound address — all visible at a glance.

One-Click Kill

Select a process, confirm, SIGTERM. No terminal, no sudo, no guessing which PID to kill.

Auto-Refresh

Polls every 2 seconds. New ports highlighted in green. Closed ports disappear. Always current.

Local & System Tabs

Filter between your dev servers and system processes.

Process Details

PID, user, protocol badge, bound address — all in monospace.

Kill Confirmation

Modal with full process details before termination. No accidental kills.

Menu Bar Native

Lives in the menu bar. No Dock icon. No window. Always accessible.

Brutalist UI

Hard borders, monospace data, red kill buttons.

Zero Dependencies

Pure Swift 6 + SwiftUI. No Homebrew, no frameworks.

macOS 14+

Built for modern macOS with strict concurrency.
